Audio-driven heightfield water simulation

An interactive real-time water surface simulation where wave energy is injected using audio analysis and direct user interaction.

Overview

This project explores artistic and interactive water surface simulation using a heightfield-based wave model. The focus is not strict physical accuracy, but stability, responsiveness, and emergent patterns when driven by audio signals.

Features

  • Real-time heightfield wave propagation
  • Audio-driven excitation using FFT analysis
  • Interactive mouse-based wave injection
  • Artist-friendly parameter controls
  • Multiple visualization modes

Tech stack

C++17 · OpenGL · GLSL · ImGui · FFT-based audio analysis
